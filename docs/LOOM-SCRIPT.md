# 2-Minute Loom Script — Aethelred RWA

**Target length:** 2:00 – 2:15  
**Record:** Screen + voice (Loom or similar)  
**Before recording:** Phantom on **devnet**, site open at https://rwa.buildingcultureid.space

---

## Setup (before you hit record)

1. Close unrelated tabs; full-screen browser
2. Wallet: **Solana devnet**, small SOL balance
3. Optional: pre-approve KYC on a test wallet so purchase works live
4. Open tabs in order (or use one tab and navigate):
   - Homepage
   - `/explore` → Berggasse 35
   - `/about`
   - GitHub repo (optional end card)

- **Skip** perp markets, Kamino lend, admin panel — not enough time
- **If purchase fails live:** say "settlement runs on devnet" and show Portfolio with existing holdings
- **Title the video:** `Aethelred RWA — Solana Grant Demo (2 min)`
