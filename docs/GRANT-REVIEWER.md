## Recommended review path (~5 min)

1. Watch **2-min Loom** (on `/grant` when `VITE_LOOM_DEMO_URL` is set) or follow [LOOM-SCRIPT.md](./LOOM-SCRIPT.md)
2. Walk [DEMO.md](./DEMO.md) on devnet (Phantom → Explore → KYC → purchase)
3. Verify [ON-CHAIN.md](./ON-CHAIN.md) program IDs on Solana Explorer
4. Read [ARCHITECTURE.md](./ARCHITECTURE.md) + [security-audit-v1.md](./security-audit-v1.md)
5. Full application: [GRANT.md](../GRANT.md) and [SOLANA-GRANT-FORM.md](./SOLANA-GRANT-FORM.md)
